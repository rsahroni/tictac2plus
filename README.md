# 🕹️ TicTac2Plus

> **TicTac2Plus** is a strategic evolution of the classic Tic Tac Toe, designed with a modern neon aesthetic and dynamic gameplay mechanics.

---

## ✨ Features

- **🎨 Customizable Colors**: Choose your vibe! Select from Red, Blue, Green, or Yellow pawns.
- **⚔️ Local Multiplayer**: Competitive 1v1 strategy on a single device.
- **🔄 Two-Phase Gameplay**: 
  - **DROP Phase**: Strategically place your pawns on the board.
  - **MOVE Phase**: Once all pawns are placed, slide them across the board to outsmart your opponent!
- **🌈 Neon Glow UI**: A sleek, dark-themed interface with glowing visual effects for a premium feel.
- **📱 Built with React Native & Expo**: Cross-platform ready and smooth performance.

---

## 🛠️ Tech Stack

- **Framework**: [React Native](https://reactnative.dev/)
- **Runtime**: [Expo](https://expo.dev/)
- **Styling**: Vanilla StyleSheet with shadow/glow effects.

---

## 🚀 Getting Started

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) and [Expo Go](https://expo.dev/expo-go) (on your mobile device) installed.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/tictac2plus.git
   cd tictac2plus
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npx expo start
   ```

4. **Scan the QR Code**: Use your camera (iOS) or the Expo Go app (Android) to run the game on your physical device.

---

## 🎮 How to Play

1. **The Goal**: Align 3 of your colored pawns (horizontally, vertically, or diagonally) to win!
2. **Phase 1 (DROP)**: Players take turns placing their 3 pawns on the 3x3 grid.
3. **Phase 2 (MOVE)**: Once all pawns are on the board, players must move one of their pawns to an adjacent empty spot on each turn.
4. **Win**: Be the first to form a line of 3!
